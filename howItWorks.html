How does it work? Simple.

This page uses a javascript library called Ping.js.

Since Javascript does not have a native way to send pings due to XHR requests being affected by cross-domain issues (CORS), they are blocked by default.

Ping.js uses a method of loading a host's favicon.ico image, and recording it's response time. If the favicon image does not exist, it will
print an error along with it's ping value (recorded in ms).


Color Meaning Numbers
Green Latency to server is good. 0 - 70ms
Orange Latency to server is decent. 71 - 100ms
Red Latency to server is high. > 100ms

For more accurate results, ping it at least 5 times - and then average those numbers.
